# 插入

sqala支持实体对象直接生成插入语句：

// 省略实体类定义
val people: People = ???

val i = insert(people)

支持使用对象列表批量写入：

val peopleList: List[People] = ???

val i = insert(peopleList)

使用autoInc标记的字段会在生成SQL时跳过。

当然我们也可以使用DSL来构建INSERT语句，这样就可以更精细地指定插入字段：

val i1 = insert[People](p => (p.id, p.name)) values (1, "小黑")

val i2 = insert[People](p => (p.id, p.name)) values List((1, "小黑"), (2, "小白"))

# 更新

对于使用primaryKey或autoInc标记好主键字段的实体对象，可以直接生成按主键更新的SQL：

// 省略实体类定义
val people: People = ???

val u = update(people)

可以额外传入一个skipNone参数，则会则更新时跳过值为None的字段：

val u = update(people, skipNone = true)

我们也可以使用DSL来构建UPDATE语句：

val u = update[People].set(p => p.name := "小黑").set(p => p.gender := Gender.Male).where(_.id == 1)

:=右侧不仅可以是简单的值，也可以是其它表达式：

val u = update[A].set(a => a.x := a.x + 1)

# 删除

我们可以使用DSL来构建DELETE语句：

val d = delete[People].where(_.id === 1)

# 按主键插入或更新

我们可以使用save方法来生成使用主键插入或更新数据的SQL语句，前提是实体类上配置好了主键字段：

val people: People = ???

val s = save(people)
